## The Project
The project main goal is build software testing of each system aspect and automatize, in Java, two or more test cases for analisys for the following setting: A software for craftsman work pricing build on a spreadsheet. The system must be capable of calculate the selling price of all itens registered by the craftman. The user can register undetermided number of itens.
