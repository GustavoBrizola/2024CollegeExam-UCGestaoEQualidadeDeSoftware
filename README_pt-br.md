## O Projeto
O objetivo do trabalho é desenvolver planos de testes sobre cada aspecto do sistema e automatizar, em Java, pelo menos dois casos de testes para análise. tendo em mente um software para precificação do trabalho de artesãos. Com base em uma planilha de cálculos, o sistema deve ser capaz de apresentar o valor de venda de cada item cadastrado pelo artesão. O artesão poderá cadastrar “n” itens que poderão ser mantidos ao longo do tempo.
